Color real en hexadecimal

Color filtrado con css filter

El filtro sólo funciona si el color base es negro.

El objetivo es usar éstos filtros para poder cambiar el color de los íconos.

Código original de Barrett Sonntag

El mismo está basado en éste generador de filtros para rgb

Si el set de íconos no es negro puedes anteponer "brightness(0) saturate(100%)" a tus propiedades de filtros que convertiran al set de íconos a negro.
